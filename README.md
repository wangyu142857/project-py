# project-py

这是一个面向“前端开发者转 Python 全栈开发者”的成长型仓库。

仓库目标不是只存放零散笔记，而是逐步沉淀：

- 学习文档
- 单点示例
- 完整项目实践
- 工程化能力
- 部署与交付经验

## 目录结构

```text
project-py/
├── docs/         # 学习大纲、模块文档、路线说明
├── examples/     # 按主题拆分的示例代码
├── apps/         # 完整项目实践
├── src/          # 主应用代码
├── tests/        # 测试代码
├── scripts/      # 辅助脚本
├── migrations/   # 数据库迁移
├── docker/       # Docker 相关配置
└── deploy/       # 部署相关配置与说明
```
