# 01 Python Foundations

## 模块目标

建立 Python 后端开发所需的语言基础、工程基础和命令行习惯，为后续 FastAPI、数据库和工程化学习做准备。

## 学习重点

- Python 基础语法与数据类型
- 函数、类、模块、包
- 异常处理
- 文件操作
- 迭代器、生成器、装饰器
- 类型注解
- 虚拟环境与依赖管理
- 基础命令行与 Git 使用

## 实践任务

- 编写若干 Python 小练习
- 搭建一个最小 Python 项目结构
- 使用虚拟环境管理依赖
- 编写一个简单的命令行脚本

## 完成标准

- 能独立阅读和编写基础 Python 代码
- 能理解模块组织方式
- 能完成最基本的项目初始化
