# 02 HTTP REST Auth

## 模块目标

理解 Web 后端的通信基础和认证基础，建立从请求进入服务到响应返回浏览器的完整认知。

## 学习重点

- HTTP / HTTPS 基础
- 请求方法、状态码、请求头
- RESTful API 设计
- JSON、表单、文件上传
- Cookie、Session、Token
- JWT
- CORS
- 分页、过滤、排序、幂等性

## 实践任务

- 设计一组简单 REST API
- 手动分析一次完整请求过程
- 实现一个最小 JWT 登录示例
- 编写接口参数与返回结构说明

## 完成标准

- 能解释常见 HTTP 概念
- 能设计基础 REST API
- 能理解并实现基本认证流程
