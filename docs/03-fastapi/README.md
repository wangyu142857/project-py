# 03 FastAPI

## 模块目标

掌握使用 FastAPI 构建现代 Python API 服务的核心能力，并具备从路由到异常处理的完整开发能力。

## 学习重点

- 路由与项目结构
- 请求参数与响应模型
- Pydantic 校验
- 依赖注入
- 中间件
- 异常处理
- 文件上传
- OpenAPI / Swagger 文档

## 实践任务

- 初始化一个 FastAPI 项目
- 编写用户与 Todo 示例接口
- 为接口增加参数校验和错误处理
- 输出接口文档并本地联调

## 完成标准

- 能独立编写基础 API
- 能组织清晰的项目结构
- 能处理常见输入校验和异常场景

## 后续可补充

- 项目脚手架
- 路由设计规范
- 常用依赖封装
- 中间件示例
