# 04 PostgreSQL And SQL

## 模块目标

建立关系型数据库基础，重点掌握 SQL 语句能力和 PostgreSQL 的常见使用方式。

## 学习重点

- 数据库基础概念
- 建表与字段设计
- 增删改查
- 条件过滤
- 排序与分页
- 多表关联
- 聚合查询
- 索引与事务

## 实践任务

- 设计博客或 Todo 的数据库表
- 编写核心 SQL 查询
- 为典型查询增加索引
- 分析简单查询性能

## 完成标准

- 能设计基础数据表
- 能写出常见业务 SQL
- 能理解索引和事务的基本作用
