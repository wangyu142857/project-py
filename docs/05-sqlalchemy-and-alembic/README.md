# 05 SQLAlchemy And Alembic

## 模块目标

掌握 Python 项目中的 ORM 使用方式和数据库迁移流程，建立模型层与数据库演进能力。

## 学习重点

- SQLAlchemy 模型定义
- session 管理
- 基础查询与复杂查询
- 一对多、多对多关系
- 数据写入与更新
- Alembic 迁移
- schema 变更管理

## 实践任务

- 将数据库表映射为 ORM 模型
- 完成一组 CRUD 操作
- 添加关联关系
- 创建并执行一次迁移

## 完成标准

- 能在项目中使用 ORM 处理主要数据访问逻辑
- 能独立完成基础迁移流程
- 能理解模型与数据库结构的对应关系
