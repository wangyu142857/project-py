# 06 Auth And Authorization

## 模块目标

建立完整的用户系统、登录流程和权限控制能力，能够支持前后端分离项目中的认证与授权设计。

## 学习重点

- 用户注册与登录
- 密码加密
- JWT 鉴权
- 刷新 Token
- RBAC 权限模型
- 接口权限控制
- 角色与资源边界

## 实践任务

- 实现注册、登录、退出
- 实现当前用户信息接口
- 为不同角色配置权限
- 为关键接口增加访问控制

## 完成标准

- 能独立实现基础用户系统
- 能解释认证与授权的区别
- 能设计简单但可用的权限模型
