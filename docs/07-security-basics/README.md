# 07 Security Basics

## 模块目标

建立后端开发中最基本的安全意识，理解常见风险及其基础防护方式。

## 学习重点

- SQL 注入
- XSS
- CSRF
- 密码安全
- 权限越权
- 文件上传风险
- 敏感信息管理
- 安全配置基础

## 实践任务

- 梳理常见接口安全风险
- 为登录和上传场景补充防护措施
- 检查配置和敏感信息管理方式
- 整理项目安全检查清单

## 完成标准

- 能识别常见安全风险
- 能在项目中加入基础安全防护
- 能避免明显的安全错误
