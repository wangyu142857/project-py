# 09 Redis And Celery

## 模块目标

掌握缓存和异步任务的基础使用场景，理解如何提升性能和拆分慢操作。

## 学习重点

- Redis 基础数据结构
- 缓存设计
- 验证码与会话场景
- 限流基础
- Celery 基础
- 异步任务处理
- 定时任务

## 实践任务

- 为接口增加缓存
- 使用 Redis 存储验证码或会话数据
- 增加一个异步任务
- 分析同步与异步处理的取舍

## 完成标准

- 能识别适合使用缓存的场景
- 能实现简单异步任务
- 能理解请求链路中的慢操作优化思路
