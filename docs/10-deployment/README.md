# 10 Deployment

## 模块目标

建立项目部署上线能力，能够将本地开发完成的服务运行到服务器环境中。

## 学习重点

- Linux 基础
- Docker
- Docker Compose
- Nginx
- Uvicorn / Gunicorn
- 域名与 HTTPS
- 环境隔离
- 基础 CI/CD

## 实践任务

- 为项目编写 Docker 配置
- 本地通过 Docker Compose 启动服务
- 使用 Nginx 做反向代理
- 整理部署步骤文档

## 完成标准

- 能独立完成本地容器化运行
- 能完成基础部署
- 能解释服务上线的主要组成部分

## 后续可补充

- 部署脚本
- 运维笔记
- CI 配置
- 上线检查表
